selectableList
service in module ng-puremodels

Description

wrap array with select-unselect-event on select functionality selection can be 'single' : only one index can be selected, and 'multi' when number of elements are selected.

Two modules of selection are not affecting each other.

Methods

  • getList()

    return array of objects

  • getSelectedIndex()

    return selected index in the array, or -1 if nothing selected

  • getSelectedObject()

    return selected object in the array, or undefined if nothing selected

  • multiSelect(index)

    select object in multi selection

    Parameters
    ParamTypeDetails
    indexinteger

    value of index to select

  • multiToggleSelect(index)

    toggleSelect selection on object

    Parameters
    ParamTypeDetails
    indexinteger

    value of index to select

  • multiUnselect(index)

    unselect object in multi selection

    Parameters
    ParamTypeDetails
    indexinteger

    value of index to select

  • selectAll()

    select all objects

  • selectIndex(index)

    change selected index and fire event if index was changed

    Parameters
    ParamTypeDetails
    indexinteger

    value of index to select

  • toggleAll()

    toggle all objects multi selection

  • unselectAll()

    unselect all objects

  • unselectIndex()

    unselect index and fire event if index was changed

  • unselectIndex()

    toggle selection and fire event if index was changed

Properties

  • fireChangeMultiSelectionEvent

  • fireChangeSelectionEvent

    function to invoke on change selection event

  • indexOf
